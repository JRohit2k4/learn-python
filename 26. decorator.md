## Decorator :  
- A function that extends the behaviour of another function.
- Without modifying the base function.
- Pass the base function as an argument to the decorator.

```python
def add_sprinkler(func):
    def wrapper(*args, **kwargs):
        print("sprinklers added to your ice-cream")
        func(*args, **kwargs)
    return wrapper

def add_fudge(func):
    def wrapper(*args, **kwargs):
        print("fudge added to your ice-cream")
        func(*args, **kwargs)
    return wrapper

@add_sprinkler
@add_fudge
def ice_cream(flavor):
    print(f"Here is your {flavor} ice-cream.")

ice_cream("chocolate")
```
```
Output:
sprinklers added to your ice-cream
fudge added to your ice-cream
Here is your chocolate ice-cream.
```
